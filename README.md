# jaxx-vue-auth

## Project setup
```
yarn install
```
Rename ./public/index.example.html to index.html and provide your API url as "VUE_APP_API_BASE_URL"!

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
